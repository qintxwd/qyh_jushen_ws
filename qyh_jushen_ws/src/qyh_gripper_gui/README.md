# JODELL EPG Gripper GUI

PyQt5 GUI for dual gripper control and monitoring.

## Features

- Real-time state monitoring for both left and right grippers
- Activation control
- Precise position, speed, and force control
- Quick action buttons (open, close, soft grip, etc.)
- Force feedback visualization
- Object detection status

## Usage

```bash
ros2 run qyh_gripper_gui gripper_gui
```

## Requirements

- PyQt5
- rclpy
- qyh_gripper_msgs
